# **App Name**: SynergyHR Portal

## Core Features:

- User Registration: Allows admins and HR to register new users with specific roles (admin, HR, employee).
- Secure Login: Enables users to securely log in to the portal using email and password.
- Role-Based Access Control: Implements RBAC to restrict access to features based on user roles, such as admin, HR, and employee. Implemented using the middleware file provided in the documentation
- Profile Management: Allows users to view and edit their profile information.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), symbolizing trust and professionalism.
- Background color: Light gray (#F9FAFB), provides a clean and modern look.
- Accent color: Soft orange (#F59E0B) to highlight important actions and sections.
- Body font: 'Inter', a sans-serif font, provides a clean and modern reading experience.
- Headline font: 'Space Grotesk' for impactful and tech-forward headers. Use 'Inter' for longer body text.
- Use minimalist icons from a consistent set, in blue or orange, for quick navigation.
- Modern and clean layout with clear sections for easy navigation.
- Subtle animations for a polished user experience when elements load or transitions occur.
